Self-Reflective RAG (Agentic RAG)
* Goal: Use LangGraph to implement a RAG system that evaluates the relevance of retrieved documents and decides whether to search again or rely on the LLM.
* Components: LangGraph, tavily-python (for search), CRAG (Corrective RAG) techniques.
* Key Learnings: Building autonomous agents that can correct their own mistakes (hallucination reduction).
